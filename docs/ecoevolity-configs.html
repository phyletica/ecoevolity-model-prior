The ecoevolity configsÂ¶

The ecoevolity configuration files in the ecoevolity-configs directory are core to this project. They specify the models that we will be using to simulate and analyze genome-scale datasets.

By design, all of these configs are identical except for the prior on the event (divergence) model. Remember, the goal of the project is to compare different ways of modeling shared divergence events.
